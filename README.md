```
npx github:mfredlundh/setup

  github:mfredlundh/boom
Ok to proceed? (y) y
  ✔ Install development apps
  ✔ Install productivity apps
  ❯ Install browsers
    ✔ Firefox
    ⠦ Google Chrome
    ✔ Brave
    Install communication
```

## Requirements
- NodeJS
- Brew

## Usage
```
npx github:mfredlundh/setup
```
